API Key Tips
Usage in REST API

Include the key in your request JSON:

{
  "api_key": "dgf-xxxxxxxxxxxxxxxx",
  "request_type": "ECHO",
  "payload": { "message": "hello" }
}
Usage in WebSocket

Send the same JSON structure over the WebSocket connection at ws://host:port/ws/gateway.

Multiple Keys per User

A user can have multiple API keys for different environments (dev, staging, production) or services. Each key authenticates as the same user.

Key Revocation

Delete a key from the API Keys list to immediately revoke access. The admin master key (dgf-admin-key-0001) cannot be deleted via the UI.

Security Note

Keys are stored in plaintext in config/apikeys.json. Protect this file with OS-level permissions. Never embed keys in source code — use environment variables or secrets managers.
