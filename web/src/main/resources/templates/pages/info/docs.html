API Documentation

POST /api/v1/request

Submit a request for handler execution.

{
  "api_key": "dgf-admin-key-0001",
  "request_type": "ECHO",
  "payload": { ... },
  "delivery_destination": "optional-topic",
  "ttl_minutes": 30
}
GET /api/v1/handlers

Returns registered request types and count.

GET /api/v1/health

Health check with handler count and recent execution count.

GET /api/v1/status

Returns recent handler execution states.

POST /api/v1/reload

Hot-reload handler configurations from disk.
