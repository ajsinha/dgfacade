About Ingesters

An Ingester binds to an Input Channel to consume messages from a messaging broker (Kafka, ActiveMQ, FileSystem, etc.) and submit them into DGFacade's request processing engine. Each ingester references an input channel by ID, which in turn references a broker for connection details. Ingester configs are stored in config/ingesters/.
